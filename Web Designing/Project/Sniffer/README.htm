Program Details

I started this program in order to study the relationships of various network protocols.

This program depends on winpcap and the following Jpcap wrapper library: http://netresearch.ics.uci.edu/kfujii/jpcap/doc/index.html. I have found this library to be much easier to use then the soureforge jpcap project. That project is also dead it appears as well. You could port this program to linux by using libpcap. Winpcap is simply the windows version of this library.
